# Hash Tables, Binary Trees, Sorting algorithms & Big O

## 1. Hash Tables

A **hash table** is a data structure that implements an associative array

### Key Concepts:
**Hash Function**: 
- Converts a key into an array index
**Collision Handling**: 
- Chaining: Store colliding elements in a linked list
- Open Addressing: Find Alternative empty slots
**Load**: 
- Ratio of entries to array size

### Advantages:
- Fast case lookups
- Efficient for caching and indexing
- Constant time operations on average
